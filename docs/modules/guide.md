## 기능 요구사항

| ID | 요구사항 | 상태 |
|----|----------|------|
| FR-100 | **가이드 페이지 렌더링** (`/guide`): 몽골어 성무일도 총지침(서문·소개 발췌·기도시간 전례 지침 개요)을 구조화된 형태로 표시한다. 목차에서 각 섹션으로 앵커 이동이 가능하다. **범위:** 몽골어 원서 p.8–20. 원서 p.22 이후의 "Залбиралт цагийн ёслолын дэг жаяг"(각 시기별 공통문 본문)은 본 페이지가 아닌 `/pray/[date]/[hour]` 기도 조립이 제공한다. | 완료 |
| FR-101 | **가이드 목차(Гарчиг)**: 3개 주요 섹션 + 소개 8개 소항목(§1, §2, §5, §6, §8, §9, §10, §11 — 몽골어 축약본은 §3, §4, §7을 생략) + 지침 3개 소항목 + Зүүлт тайлбар 링크. 각 항목에 원서 PDF 페이지 번호(х.N) 표시. 클릭 시 해당 섹션으로 스크롤. | 완료 |
| FR-102 | **가이드 각주**: 소개 본문의 성경/공의회 문헌 각주 28개(번호 `1, 2, 30–48, 52–58`)를 하단에 표시한다. 본문의 `[N]` 링크(`#fnref-N`) 클릭 시 `#fn-N` 각주로 이동하고, 각주의 `[N]` 링크(`#fn-N → #fnref-N`)로 본문으로 복귀할 수 있어야 한다(양방향 이동). | 완료 |

## 데이터 구조 (gilh.json)

```
{
  "title": string,
  "sections": [
    {
      "id": string,           // 앵커 ID
      "title": string,        // 몽골어 섹션 제목
      "page": number,         // PDF 페이지 번호
      "paragraphs"?: string[],  // 서문: 직접 문단
      "subtitle"?: string,      // 소개: 부제
      "subsections"?: [         // 소개/루브리카: 하위 섹션
        {
          "id": string,
          "number"?: string,    // GILH 조항 번호 (§1, §2, ...)
          "title": string,
          "page"?: number,
          "paragraphs": string[]
        }
      ]
    }
  ],
  "footnotes": [
    { "number": string, "text": string }
  ]
}
```

## 스코프 제한

원서 p.22 이후의 **"Залбиралт цагийн ёслолын дэг жаяг"**(아침·저녁·끝기도 공통문, Invitatory 시편 95/100/67/24, Benedictus·Magnificat·Nunc Dimittis, 시기별 гуйлтын залбирал 등 매일 전례의 실제 본문)은 본 가이드 페이지의 범위가 아니다. 이 공통문은 [hour-assembly](hour-assembly.md) 모듈이 런타임에 조립하여 `/pray/[date]/[hour]` 페이지로 제공한다.
